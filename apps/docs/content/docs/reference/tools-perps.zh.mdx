import { TypeTable } from "fumadocs-ui/components/type-table";

永续合约工具支持两个提供商：Hyperliquid（EVM 地址，主网和测试网）和 Jupiter Perps（Solana 地址，仅主网）。省略时默认使用 Hyperliquid。写入工具需要[策略审批](/zh/docs/reference/tools-policy)。

## get_market_data

返回所选提供商的市场元数据。可返回所有市场或单个市场。

### 参数

<TypeTable
  type={{
    provider: {
      type: '"hyperliquid" | "jupiter_perps"',
      description: "永续合约提供商。",
      default: '"hyperliquid"',
    },
    market: {
      type: "string",
      description: "市场符号（例如 'BTC'）。若省略，返回所有市场。",
    },
  }}
/>

### 响应

```json
{
  "provider": "hyperliquid",
  "markets": [
    {
      "coin": "BTC",
      "mark_px": "64250.00",
      "funding_rate": "0.0001",
      "open_interest": "150000000"
    }
  ]
}
```

响应结构仅供参考，实际字段可能有所不同。

### 示例

<Tabs items={["All markets", "Single market"]}>
  <Tab value="All markets">
    ```json
    {
      "provider": "hyperliquid"
    }
    ```
  </Tab>
  <Tab value="Single market">
    ```json
    {
      "provider": "hyperliquid",
      "market": "BTC"
    }
    ```
  </Tab>
</Tabs>

### 注意事项

- 如果 Hyperliquid API 暂时不可用，Seashail 可能返回缓存快照。

## get_positions

返回所选提供商和钱包的当前持仓状态。

### 参数

<TypeTable
  type={{
    provider: {
      type: '"hyperliquid" | "jupiter_perps"',
      description: "永续合约提供商。",
      default: '"hyperliquid"',
    },
    wallet: {
      type: "string",
      description: "若省略，使用活跃钱包。",
    },
    account_index: {
      type: "integer",
      description: "若省略，使用活跃账户索引。",
    },
  }}
/>

### 各场所行为

<Tabs items={["Hyperliquid", "Jupiter Perps"]}>
  <Tab value="Hyperliquid">
    - 使用钱包的 **EVM 地址** - 如果选定的钱包/账户没有 EVM 地址，调用失败 -
    主网和测试网均可使用
  </Tab>
  <Tab value="Jupiter Perps">
    - 使用钱包的 **Solana 地址** - 仅主网（网络模式必须为主网）
  </Tab>
</Tabs>

### 响应

```json
{
  "provider": "hyperliquid",
  "positions": [
    {
      "market": "BTC",
      "side": "long",
      "size": "0.5",
      "entry_px": "63000.00",
      "mark_px": "64250.00",
      "unrealized_pnl": "625.00",
      "leverage": 3
    }
  ]
}
```

响应结构仅供参考，实际字段可能有所不同。

### 示例

<Tabs items={["Hyperliquid", "Jupiter Perps"]}>
  <Tab value="Hyperliquid">
    ```json
    {
      "provider": "hyperliquid",
      "wallet": "my-wallet",
      "account_index": 0
    }
    ```
  </Tab>
  <Tab value="Jupiter Perps">
    ```json
    {
      "provider": "jupiter_perps",
      "wallet": "my-wallet",
      "account_index": 0
    }
    ```
  </Tab>
</Tabs>

## open_perp_position

开设永续合约仓位。需要策略审批。

### 参数

<TypeTable
  type={{
    provider: {
      type: '"hyperliquid" | "jupiter_perps"',
      description: "永续合约提供商。",
      default: '"hyperliquid"',
    },
    wallet: {
      type: "string",
      description: "若省略，使用活跃钱包。",
    },
    account_index: {
      type: "integer",
      description: "若省略，使用活跃账户索引。",
    },
    market: {
      type: "string",
      description: "市场符号（例如 'BTC'、'ETH'、'SOL'）。",
      required: true,
    },
    side: {
      type: '"long" | "short"',
      description: "仓位方向。",
      required: true,
    },
    size: {
      type: "string",
      description: "仓位大小。",
      required: true,
    },
    size_units: {
      type: '"usd" | "asset"',
      description: "Jupiter Perps 仅支持 'usd'。",
      default: '"usd"',
    },
    leverage: {
      type: "integer",
      description: "杠杆倍数。",
      required: true,
    },
    order_type: {
      type: '"market" | "limit"',
      description: "Jupiter Perps 仅支持 'market'。",
      default: '"market"',
    },
    limit_px: {
      type: "string",
      description: "order_type='limit' 时必填。限价。",
    },
    slippage_bps: {
      type: "integer",
      description: "滑点容忍度，单位基点。",
      default: "50",
    },
    usd_value: {
      type: "number",
      description: "策略评估用的估算美元价值。若省略，视为未知。",
    },
    usd_value_known: {
      type: "boolean",
      description: "可选覆盖。若提供了 usd_value，则视为 true。",
      default: "false",
    },
  }}
/>

### 各场所行为

<Tabs items={["Hyperliquid", "Jupiter Perps"]}>
  <Tab value="Hyperliquid">
    - 支持**市价**和**限价**单 - 支持 size_units：**"usd"** 或 **"asset"** -
    使用钱包的 **EVM 地址** - 主网和测试网均可使用
  </Tab>
  <Tab value="Jupiter Perps">
    - 仅支持**市价单** - 仅支持 size_units：**"usd"** - 使用钱包的 **Solana
    地址** - 仅主网（网络模式必须为主网）
  </Tab>
</Tabs>

### 响应

```json
{
  "provider": "hyperliquid",
  "market": "BTC",
  "side": "long",
  "size": "100",
  "size_units": "usd",
  "leverage": 3,
  "order_type": "market",
  "signature": "0x...",
  "status": "confirmed"
}
```

响应结构仅供参考，实际字段可能有所不同。

### 示例

<Tabs items={["Hyperliquid market", "Hyperliquid limit", "Jupiter Perps market"]}>
  <Tab value="Hyperliquid market">
    ```json
    {
      "provider": "hyperliquid",
      "wallet": "my-wallet",
      "account_index": 0,
      "market": "BTC",
      "side": "long",
      "size": "100",
      "size_units": "usd",
      "leverage": 3,
      "order_type": "market",
      "slippage_bps": 50
    }
    ```
  </Tab>
  <Tab value="Hyperliquid limit">
    ```json
    {
      "provider": "hyperliquid",
      "wallet": "my-wallet",
      "account_index": 0,
      "market": "ETH",
      "side": "short",
      "size": "50",
      "size_units": "usd",
      "leverage": 2,
      "order_type": "limit",
      "limit_px": "2500"
    }
    ```
  </Tab>
  <Tab value="Jupiter Perps market">
    ```json
    {
      "provider": "jupiter_perps",
      "wallet": "my-wallet",
      "account_index": 0,
      "market": "SOL",
      "side": "long",
      "size": "50",
      "size_units": "usd",
      "leverage": 5,
      "order_type": "market",
      "slippage_bps": 50
    }
    ```
  </Tab>
</Tabs>

### 注意事项

- 需要[策略审批](/zh/docs/reference/tools-policy)——交易可能被自动批准、需要确认或被阻止，取决于策略配置和美元价值。
- Jupiter Perps 目前仅支持**市价单**和 **size_units="usd"**。

## close_perp_position

平仓永续合约仓位。需要策略审批。

### 参数

<TypeTable
  type={{
    provider: {
      type: '"hyperliquid" | "jupiter_perps"',
      description: "永续合约提供商。",
      default: '"hyperliquid"',
    },
    wallet: {
      type: "string",
      description: "若省略，使用活跃钱包。",
    },
    account_index: {
      type: "integer",
      description: "若省略，使用活跃账户索引。",
    },
    market: {
      type: "string",
      description: "市场符号（例如 'BTC'）。",
      required: true,
    },
    side: {
      type: '"long" | "short"',
      description: "可选方向提示；平仓方向从当前仓位推导。",
    },
    size: {
      type: "string",
      description: "可选平仓大小（美元或资产单位）。若省略，全部平仓。",
    },
    size_units: {
      type: '"usd" | "asset"',
      description: "大小单位。",
      default: '"asset"',
    },
    slippage_bps: {
      type: "integer",
      description: "滑点容忍度，单位基点。",
      default: "50",
    },
    usd_value: {
      type: "number",
      description: "策略评估用的估算美元价值。若省略，视为未知。",
    },
    usd_value_known: {
      type: "boolean",
      description: "可选覆盖。若提供了 usd_value，则视为 true。",
      default: "false",
    },
  }}
/>

### 各场所行为

<Tabs items={["Hyperliquid", "Jupiter Perps"]}>
  <Tab value="Hyperliquid">
    - 支持**部分平仓**（提供 `size`）或**全部平仓**（省略 `size`）- 支持
    size_units：**"usd"** 或 **"asset"** - 使用钱包的 **EVM 地址**
  </Tab>
  <Tab value="Jupiter Perps">
    - 仅支持**全部平仓**（省略 `size`）- 若省略 `side`，Seashail 从持仓中推断 -
    使用钱包的 **Solana 地址** - 仅主网
  </Tab>
</Tabs>

### 响应

```json
{
  "provider": "hyperliquid",
  "market": "BTC",
  "side": "long",
  "signature": "0x...",
  "status": "confirmed"
}
```

响应结构仅供参考，实际字段可能有所不同。

### 示例

<Tabs items={["Hyperliquid full close", "Hyperliquid partial close", "Jupiter Perps full close"]}>
  <Tab value="Hyperliquid full close">
    ```json
    {
      "provider": "hyperliquid",
      "wallet": "my-wallet",
      "account_index": 0,
      "market": "BTC",
      "slippage_bps": 50
    }
    ```
  </Tab>
  <Tab value="Hyperliquid partial close">
    ```json
    {
      "provider": "hyperliquid",
      "wallet": "my-wallet",
      "account_index": 0,
      "market": "ETH",
      "size": "0.5",
      "size_units": "asset",
      "slippage_bps": 50
    }
    ```
  </Tab>
  <Tab value="Jupiter Perps full close">
    ```json
    {
      "provider": "jupiter_perps",
      "wallet": "my-wallet",
      "account_index": 0,
      "market": "SOL",
      "slippage_bps": 50
    }
    ```
  </Tab>
</Tabs>

### 注意事项

- 需要[策略审批](/zh/docs/reference/tools-policy)。
- Jupiter Perps 目前仅支持平仓**整个**仓位（省略 `size`）。
- 对于 Jupiter Perps，若省略 `side`，Seashail 会推断持仓是多头还是空头（如果有歧义则报错）。
