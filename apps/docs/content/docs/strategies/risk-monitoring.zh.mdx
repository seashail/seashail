## 概述

建立风险监控例程：检查投资组合盈亏、各借贷/永续合约/预测市场的仓位健康状况，并在爆仓风险发生前识别出来。

**使用的工具：** `get_portfolio`、`get_portfolio_analytics`、`get_positions`、`get_lending_positions`、`get_prediction_positions`
**支持的链：** 所有支持的链
**完成时间：** 5-10 分钟

## 前提条件

- 在各协议中有已部署仓位的活跃钱包
- 了解健康因子（借贷）和爆仓价格（永续合约）

<Steps>

### 第一步：获取投资组合概况

获取投资组合总计数据及历史盈亏跟踪。

```json
{
  "name": "get_portfolio",
  "arguments": {
    "include_history": true,
    "history_limit": 7,
    "include_health": true
  }
}
```

**预期结果：** 投资组合快照，包含总美元价值、7 日变化以及任何持久化的仓位健康快照。

### 第二步：运行投资组合分析

按类型、链和近期活动获取美元交易量明细。

```json
{
  "name": "get_portfolio_analytics",
  "arguments": {
    "limit": 500,
    "since_ts": "2026-02-03T00:00:00Z"
  }
}
```

**预期结果：** 按交易类型（兑换、发送、借贷）和链的交易量总计。识别资金最活跃的地方。

### 第三步：检查永续合约持仓

查看杠杆敞口和爆仓风险。

```json
{
  "name": "get_positions",
  "arguments": {
    "provider": "hyperliquid"
  }
}
```

**预期结果：** 开仓仓位，包含规模、杠杆、入场价格、当前盈亏和爆仓价格。**警告信号：** 爆仓价格在当前标记价格 5% 以内的仓位。

### 第四步：检查借贷仓位

验证健康因子是否安全（保守风险容忍度下应高于 1.5）。

```json
{
  "name": "get_lending_positions",
  "arguments": {
    "chain": "base"
  }
}
```

**预期结果：** 已提供的抵押品、借入金额、健康因子。**警告信号：** 健康因子低于 1.3（1.0 时发生爆仓）。

### 第五步：查看预测市场敞口

检查活跃的 Polymarket 仓位和结果概率。

```json
{
  "name": "get_prediction_positions",
  "arguments": {
    "chain": "polygon"
  }
}
```

**预期结果：** 活跃市场仓位，包含当前概率和未实现盈亏。**注意：** 预测市场没有爆仓风险，但如果结果不利于你的仓位，可能归零。

</Steps>

## 变体

### 单链监控

专注于监控一条高活跃度的链以减少干扰信息。

```json
{
  "name": "get_portfolio",
  "arguments": {
    "chains": ["solana"],
    "include_history": true
  }
}
```

### 专注仓位监控（仅永续合约）

如果你的唯一风险敞口是永续合约，跳过投资组合概况，直接查看永续合约仓位。

```json
{
  "name": "get_positions",
  "arguments": {
    "provider": "hyperliquid"
  }
}
```

然后手动对比爆仓价格与当前市场价格。

## 注意事项

- **建立定期节奏：** 如果主动交易，每天运行本方案；被动流动性挖矿时每周运行。
- **健康因子阈值：**
  - 高于 2.0：非常安全
  - 1.5-2.0：在大多数市场条件下安全
  - 1.2-1.5：警戒区；如果波动性增加，添加抵押品或减少借款
  - 低于 1.2：高风险；需要立即采取行动
- **资金费率意识（永续合约）：** 正资金费率意味着多头向空头支付；负资金费率意味着空头向多头支付。在 `get_market_data` 中检查资金费率以了解持续成本。
- **无爆仓预警：** Seashail 不会对爆仓风险自动发出警报。你必须手动监控，或在这些工具之上构建自动化。
