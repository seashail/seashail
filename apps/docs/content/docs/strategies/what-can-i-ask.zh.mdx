import { Accordion, Accordions } from "fumadocs-ui/components/accordion";

Seashail 理解自然语言请求，并将其转化为安全、符合策略的交易。本页展示你可以请求的内容，按照你想完成的目标分组——无需技术知识。

## 投资组合与余额

<Accordions>
  <Accordion title="查看我的余额">
    **示例：**
    - "显示我的 Solana 余额"
    - "我的投资组合总价值是多少？"
    - "列出我所有的钱包"
    - "我有多少 ETH？"
    - "显示我在 Base 上的 USDC 余额"

    **使用的工具：** `get_balance`、`get_portfolio`、`list_wallets`

    **了解更多：** [链与资金](/zh/docs/guides/chains-and-funding)

  </Accordion>

  <Accordion title="跟踪投资组合表现">
    **示例：**
    - "我的投资组合这周表现如何？"
    - "显示我的交易量明细"
    - "我的 30 天盈亏是多少？"
    - "显示我的交易历史"

    **使用的工具：** `get_portfolio`（带 `include_history`）、`get_portfolio_analytics`、`get_transaction_history`

    **了解更多：** [链与资金](/zh/docs/guides/chains-and-funding)

  </Accordion>

  <Accordion title="获取充值地址">
    **示例：**
    - "给我我的 Solana 充值地址"
    - "显示在 Base 上接收 USDC 的二维码"
    - "我的 ETH 地址是什么？"
    - "我可以往这个钱包发送资金到哪里？"

    **使用的工具：** `get_deposit_info`、`get_wallet_info`

    **了解更多：** [链与资金](/zh/docs/guides/chains-and-funding)

  </Accordion>
</Accordions>
